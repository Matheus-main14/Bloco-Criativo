# Bloco-Criativo

 # Bloco Criativo

Projeto desenvolvido para a disciplina de Padrões-Web, utilizando HTML5, CSS3 e Bootstrap 5.

O objetivo do projeto foi desenvolver um site como forma de aprendizado, colocando em prática os conceitos estudados durante a disciplina.

---

 Sobre o projeto

A Bloco Criativo é um site que simula uma loja virtual de produtos artesanais.

O usuário pode:

- Visualizar a página inicial;
- Conhecer os produtos disponíveis;
- Navegar pelo catálogo;
- Visualizar detalhes de cada produto;
- Preencher um formulário de encomenda;

---

 Páginas do projeto

 Página Inicial

A página inicial apresenta o projeto e oferece acesso rápido às demais áreas do site.

Contém:

- Barra de navegação;
- Apresentação do projeto;
- Produtos em destaque;
- Informações sobre os artesãos;
- Rodapé.

---

## 📦 Catálogo

A página de catálogo reúne todos os produtos cadastrados.

Cada produto possui:

- Imagem;
- Nome;
- Categoria;
- Preço;
- Botão para visualizar os detalhes.

---

## 🛒 Encomenda

Cada produto possui sua própria página de encomenda.

Nela o usuário encontra:

- Imagem do produto;
- Descrição;
- Materiais utilizados;
- Informações adicionais;
- Formulário de encomenda.
